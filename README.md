A cool farming Minecraft mod made for HackClub

This mod adds new farming items you can access in the game through Loot Crates!!!
Loot Crates have the same crafting recipe as the jukebox but with an emerald instead of a diamond. 
You might need to have some luck to get the new mod items.

Pro Tip: Smelt corn seeds to get popcorn

INSTUCTIONS TO USE:

1st download ExtraFarmingAddOnMod-1.0.0.jar
You can get it here (https://modrinth.com/mod/extra-farming-add-on-mod) or download it off of the repo if that does not work

2nd go to %appdata% -> .minecraft -> .mods and drag in the ExtraFarmingAddOnMod-1.0.0.jar into it

3rd make sure that you are in version 1.21 in minecraft and with a fabric installation and run the mod to play it

(Remember to install fabric api)

https://www.youtube.com/watch?v=v-WNTQh_m5w for more details
